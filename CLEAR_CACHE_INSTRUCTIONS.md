# Clear Browser Cache - Instructions

## Why you're seeing the old template:
The browser might be using a cached (old) version of your JavaScript file. Follow these steps:

## Solution 1: Hard Refresh (Easiest)
1. **Chrome/Edge**: Press `Ctrl + Shift + R` or `Ctrl + F5`
2. **Firefox**: Press `Ctrl + Shift + R` or `Ctrl + F5`
3. **Safari**: Press `Cmd + Shift + R`

## Solution 2: Clear Browser Cache
1. Open browser settings
2. Clear browsing data/cache
3. Select "Cached images and files"
4. Clear data
5. Refresh the page

## Solution 3: Disable Cache (Developer Mode)
1. Open Developer Tools (F12)
2. Go to Network tab
3. Check "Disable cache"
4. Keep DevTools open while testing
5. Refresh the page

## Solution 4: Verify EmailJS Template
1. Go to https://dashboard.emailjs.com/
2. Click on **Email Templates**
3. Open template ID: `template_gn3n04j`
4. Verify it's the template you want to use
5. Make sure it's saved and active

## After clearing cache:
1. Test the contact form again
2. Check your email (matechodhis@gmail.com)
3. The email should now use the correct template
