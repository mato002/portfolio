# Programming Setup Background Image Instructions

## Current Setup
The hero section now uses a programming setup background image from Unsplash. You can replace it with your preferred image.

## Option 1: Use Unsplash (Free Stock Images)
The current image is from Unsplash - a free stock photo service. You can find similar programming setup images at:
- https://unsplash.com/s/photos/programming-setup
- https://unsplash.com/s/photos/coding-desk
- https://unsplash.com/s/photos/developer-workspace

## Option 2: Download and Use Local Image
1. Download a programming setup image from:
   - Unsplash (free): https://unsplash.com
   - Pexels (free): https://www.pexels.com/search/programming/
   - Pixabay (free): https://pixabay.com/images/search/programming/
   
2. Save it to your `images` folder (e.g., `images/programming-bg.jpg`)

3. Update the CSS files:
   - In `index.css`, replace the `background-image` URLs with:
     ```css
     background-image: url('images/programming-bg.jpg');
     ```

## Option 3: Use Other Free Stock Image Services
- **Pexels**: https://www.pexels.com/search/programming/
- **Pixabay**: https://pixabay.com/images/search/programming/
- **Freepik**: https://www.freepik.com/search?format=search&query=programming%20setup

## Recommended Image Specifications
- **Resolution**: At least 1920x1080 (Full HD) or higher
- **Format**: JPG or PNG
- **Content**: Programming workspace, code on screen, developer desk, coding setup
- **Style**: Modern, clean, professional

## Where to Update
Update the `background-image` property in `index.css` for:
1. `#header` - Hero section background
2. `nav` - Navigation bar background (when it appears on scroll)
3. `#about` (mobile) - About section background on mobile
